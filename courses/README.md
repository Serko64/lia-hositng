# Kurse bearbeiten

Lehrende bearbeiten nur diesen Ordner.

- `courses.json` pflegt die Kursliste im Portal.
- Semester kommen als Unterordner, z. B. `2026-SS/`.
- Kursdateien sind Markdown-Dateien (`.md`).
- Optionale Assets/Styles liegen unter `assets/` und `styles/`.

Beim Build wird `courses/` automatisch nach `dist/courses/` kopiert.
